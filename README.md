 # 🏠 Hyderabad Home Buddy

Hyderabad Home Buddy is a modern web application that helps users explore housing options in Hyderabad. The project is built using **React, Vite, TypeScript, and Tailwind CSS** to provide a fast, responsive, and user-friendly interface.

## 🚀 Features

- 🏡 Browse houses and property listings
- 📍 Explore locations in Hyderabad
- ⚡ Fast performance using Vite
- 🎨 Modern and responsive UI with Tailwind CSS
- 📱 Mobile-friendly design

## 🛠️ Tech Stack

- **Frontend:** React, Vite, TypeScript
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **Package Manager:** npm

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/hyderabad-home-buddy.git
```

2. Navigate to the project folder

```bash
cd hyderabad-home-buddy
```

3. Install dependencies

```bash
npm install
```

4. Run the development server

```bash
npm run dev
```

5. Open in browser

```
http://localhost:5173
```
